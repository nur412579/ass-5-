Question-1: What are the different ways to select an element in the DOM?
DOM-এ HTML উপাদান নির্বাচন করার জন্য নিম্নলিখিত পদ্ধতিগুলি ব্যবহার করা যায়ঃ document.getElementById('id') → নির্দিষ্ট id সহ উপাদান নির্বাচন করে। document.getElementsByClassName('class') → নির্দিষ্ট class সহ সব উপাদান নির্বাচন করে। document.getElementsByTagName('tag') → নির্দিষ্ট tag নামের সব উপাদান নির্বাচন করে। document.querySelector('selector') → প্রথম উপাদান নির্বাচন করে যা নির্দিষ্ট CSS সিলেক্টরের সাথে মেলে। document.querySelectorAll('selector') → সমস্ত উপাদান নির্বাচন করে যা নির্দিষ্ট CSS সিলেক্টরের সাথে মেলে।
Question-2: What is the difference between innerHTML, innerText, and textContent ?
প্রপার্টিকাজinnerHTMLHTML ট্যাগ সহ সম্পূর্ণ কন্টেন্ট পায় বা সেট করে।innerTextশুধুমাত্র দৃশ্যমান (visible) টেক্সট পায় বা সেট করে।textContentসমস্ত টেক্সট পায়, এমনকি লুকানো (display: none;) থাকলেও।
Question-3: What is event delegation in the DOM?
ইভেন্ট ডেলিগেশন হল এমন একটি কৌশল যেখানে ইভেন্ট শোনার জন্য শিশুর পরিবর্তে মূল প্যারেন্ট উপাদানকে ব্যবহার করা হয়। এটি কম মেমোরি ব্যবহার করে এবং কর্মক্ষমতা উন্নত করে।
Question-4: What is event bubbling in the DOM?
ইভেন্ট বুদবুদ হল এমন একটি প্রক্রিয়া যেখানে একটি উপাদানে ইভেন্ট ট্রিগার হলে, তা তার প্যারেন্ট এবং পূর্বপুরুষ উপাদানগুলিতে গিয়ে পৌঁছায়
Question-5: How do you create, add, and remove elements using JavaScript?
